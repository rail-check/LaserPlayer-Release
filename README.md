# LaserPlayer
激光扫描仪数据播放器

![image-20220930185018378](README.assets/image-20220930185018378.png)

## 操作说明

- `左右键` —— 以单帧步进；
- `shift+左右键` —— 以10帧步进；
- `r` —— 恢复视角；

- `打开文件夹` —— 打开包含激光器数据的文件夹下所有的文件，激光器编号自动从文件名中提取，相同编号的激光器数据会拼接到一起；
- `打开文件` —— 单开单独的激光数据文件。
